## 🎨 核心視覺與產品特色

*   **🖥️ 1920 × 1080 滿版沉浸**：擺脫傳統手機框架與聊天氣泡的廉價感，轉化為大螢幕精緻電子書風格，字體專注採用 **源雲明體**。
*   **🌓 日夜雙色切換**：內建極致舒適的深色模式（墨黑底暗紅微光）與清醒的淺色模式，並自動記憶玩家的視覺偏好。
*   **🎵 虛空環境音 (BGM)**：開場搭配深空暗黑環境音樂，隨著玩家點擊「觸碰進入實相」優雅淡入，拉滿宏大而神祕的氛圍。
*   **📉 履歷式動態狀態欄**：將 AI 生成的「信任值、佔有慾、依賴性、男主心聲」等隱藏數值轉化為視覺卡片，且採用**卡片流無限累積**，可隨時切換查閱整個實相的流轉進程。
*   **🛡️ 演算失敗防呆回溯**：若遇到 AI 網路不穩或額度耗盡，系統會自動移除畫面的錯誤紀錄、扣回 Prompt、重開面板，保證遊戲體驗絕不中斷。
*   **💾 本地存檔記憶**：所有 API Key、遊玩紀錄與角色卡皆儲存於您瀏覽器的 `LocalStorage` 中，絕不經過第三方伺服器，100% 保障您的實相隱私。

## 🌎 世界觀與核心機制

在雙重性別系統的社會中，每個人都擁有生理構造（男女）與生殖階級（ABOE）兩套系統：

*   **Alpha (A)**：天生的統治者，體能智商頂尖，佔有慾強，信息素具侵略壓迫性。
*   **Beta (B)**：佔人口 80% 的中流砥柱，無信息素、無發情期，情緒最為清清醒抽離。
*   **Omega (O)**：天生的生育者，易受 Alpha 信息素吸引而陷入臣服或每月定期的發情期。
*   **Enigma (E)**：極少數的隱藏神祕性別，凌駕於 Alpha 之上，甚至能將其強制標記轉化。

### 🩸 生理交鋒鐵律
*   **腺體與信息素**：位於後頸處的敏感器官。Alpha（烈酒、硝煙、冷杉）與 Omega（甜奶、水果、花香）的信息素具有基因上的**契合匹配度**，高匹配度將引發致命吸引力。
*   **標記連結**：分為咬破腺體注入信息素的**臨時標記**，以及在高潮交配時進行高成活率灌注的**永久標記（成結綁定）**。

## 🔮 DM 主持人（敘事引擎）絕對公理

本遊戲後台注入了極其嚴格的 **【深層理解律】** 與 **【感官具象律】** 提示詞約束：
1.  **長篇巨量敘事**：DM 每次劇情推進**絕對不可低於 1500 字**。拒絕快節奏流水帳，著重於細膩的微表情、氣味交織與心理拉扯。
2.  **單一鏡頭換行**：動作、發話、內外反轉、對白與動作必須交替推進，排開多餘形容詞，保留大長句的呼吸感與視覺換行留白。
3.  **多樣性不重複**：AI 被限制每一次登場的 NPC 人名與事件盡量不重複，且匹配度作為暗自設定的變數，僅能從文字氛圍演示，不可生硬公開。
